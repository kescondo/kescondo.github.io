Selected Projects

Chef's Compass: Meal Planner

Chef's Compass saves time by helping busy professionals find recipes online, save them to their cookbook, and create meal plans from saved recipes.

Developed with a team as the final capstone project at Tech Elevator.

Sweetie Boom's Candy Shop:
E-commerce App

Sweetie Boom's Candy Shop manages inventory for a candy shop, allowing shoppers to browse available candies, add selected candies to a cart, and purchase them.

Originally created as a pair-partner project at Tech Elevator using only Java and a command line interface. My revision (shown here; currently in progress) incorporates a database and Vue.js.

PlaySnap: Toy Inventory Manager

PlaySnap organizes and keeps track of a family's toys and their details, documenting their story of play through collections, photos, and wish lists.

This personal project is currently in progress.
